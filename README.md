# PetDoctor

App that works with:
- web
- ios
- android

## Installation

1. Install Flutter Beta
2. Make sure `flutter doctor` works well
3. Build and run app with:
- web
```
./scripts/run_web.sh
```
- ios
```
./scripts/emulator_ios.sh
./scripts/run_ios.sh
```
- android
```
./scripts/emulator_android.sh
./scripts/run_android.sh
```
